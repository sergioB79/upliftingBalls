# Uplifting Balls Site

Open `index.html` in your browser. You will see a random number of pastel-colored balls of random sizes flowing around on a light green background. Click a ball to pop it and reveal an uplifting message in a handwritten font. The message will remain for a few seconds before vanishing. Only one message is shown at a time.

## Features
- Random number of balls (8-16), random sizes, random pastel colors
- Balls bounce around the screen
- Click a ball to pop it and show a message
- Message appears in the center, in a handwritten font, for 2.5 seconds
- After the message, new balls are generated

## To Use
Just open `index.html` in your browser. No server required.
